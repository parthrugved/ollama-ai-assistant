Machine Learning is a subset of Artificial Intelligence that enables computers to learn from data without being explicitly programmed. In other words, ML algorithms can improve their performance on a task over time based on experience and new data.

There are three types of Machine Learning:

1. **Supervised Learning**: The algorithm learns from labeled data, where the correct output is provided.
2. **Unsupervised Learning**: The algorithm identifies patterns in unlabeled data.
3. **Reinforcement Learning**: The algorithm learns by interacting with an environment and receiving feedback.

**Key differences between AI and ML:**

1. **Scope**: AI is a broader field that includes not only ML but also other areas like computer vision, natural language processing, and expert systems.
2. **Learning**: While AI involves creating intelligent systems, ML specifically focuses on the ability of these systems to learn from data.

**Real-world applications:**

AI and ML have many practical applications in:

1. **Image recognition**: Facial recognition, object detection, image classification.
2. **Natural Language Processing (NLP)**: Chatbots, language translation, text summarization.
3. **Predictive maintenance**: Predicting equipment failures, optimizing resource allocation.

```cpp
#include <iostream>

int main() {
    // Define the hardware register address
    uint32_t reg_addr = 0x10000000;

    // Read from the register
    uint32_t value = *(volatile uint32_t*)reg_addr;
    std::cout << "Register Value: " << value << std::endl;

    // Write to the register
    *(volatile uint32_t*)reg_addr = 0x12345678;
    return 0;
}
```
**Note:** This code snippet demonstrates accessing a hardware register using pointer arithmetic. The `volatile` keyword ensures that the compiler generates code for reading and writing the register value.
